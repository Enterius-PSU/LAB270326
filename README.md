# Задание 1



### Текст задачи



Составить список всех трехзначных чисел, в десятичной записи которых нет одинаковых цифр (операции деления и нахождения остатка от деления не использовать).  



### Алгоритм решения



1. Формируем список чисел:
2. С помощью цикла сотен и вложенных в него цикла десятков, а также и вложенного в цикл десятков цикла единиц получаем уникальные (не повторяющиеся) числа. (Проверяется равность сотен, десятков и единиц, если хотя бы один из них равен другому, то число пропускается).
3. Разворачиваем список, чтобы он начинался с начала, а не с конца.
4. Выводим список.

# Задание 2



### Текст задачи



Найти разность максимального и минимального элементов списка.



### Алгоритм решения



1. Запрашиваем количество и значения элементов списка у пользователя.
2. Рекурсивно формируем список.
3. Рекурсивно ищем минимальное и максимальное значение элемента списка. (первый элемент в каждом случае - отправная точка для поиска)
4. Выводим их значения.
5. Находим разность между ними.
6. Выводим результат.

# Задание 3



### Текст задачи



Определим множество как список без повторяющихся элементов. Для двух заданных множеств определить, является ли одно из них подмножеством другого.  



### Алгоритм решения



1. Запрашиваем количество и значения элементов для 2-ух списков у пользователя. (рекурсивно)
2. Формируем списки чисел.
3. Проверяем связь между этими списками:
4. Если один из списков пустой, то он определенно входит в другой список.
5. Иначе рекурсивно проверяем на наличие элементов одного списка в другом списке. (если находим первый элемент одного списка в другом, то проверяем следующий)
6. Выводим результат проверки.

# Задание 4



### Текст задачи



Пять детей - Алик, Боря, Витя, Лена и Даша - приехали в лагерь из 5 разных городов: Харькова, Умани, Полтавы, Славянска и Краматорска. Есть 4 высказывания: 

1\) Если Алик не из Умани, то Боря из Краматорска. 

2\) Или Боря, или Витя приехали из Харькова. 

3\) Если Витя не из Славянска, то Лена приехала из Харькова. 

4\) Или Даша приехала из Умани, или Лена из Краматорска. 

Кто откуда приехал? 



### Алгоритм решения



1. Составляем список выданных нам городов.
2. Перебираем все перестановки, сопоставляя города с детьми и проверяя 4 условия. (поиск с возвратом)
3. Если условия выполнены, то выводим результат.
